## About This Theme

Zola Pico is a minimal blog theme that combines the power of [Zola](https://www.getzola.org) static site generator with the elegant simplicity of [PicoCSS](https://picocss.com).

### Why Zola Pico?

This theme was created with a few core principles in mind:

1. **Simplicity First**: Minimal dependencies, clean code, and straightforward customization
2. **Semantic HTML**: Using proper HTML5 elements that make sense
3. **Performance**: Fast load times with no JavaScript overhead
4. **Accessibility**: Built-in support for keyboard navigation and screen readers
5. **Dark Mode**: Automatic theme switching that respects user preferences

### Technology Stack

- **Zola**: A fast static site generator written in Rust
- **PicoCSS**: A minimalist CSS framework for semantic HTML
- **Sass**: For easy style customization and theming

### Customization

You can customize this theme by:

- Modifying CSS variables in `sass/_variables.scss`
- Adding custom styles in `sass/_custom.scss`
- Overriding templates by creating files in your `templates/` directory
- Adjusting configuration in `config.toml`
